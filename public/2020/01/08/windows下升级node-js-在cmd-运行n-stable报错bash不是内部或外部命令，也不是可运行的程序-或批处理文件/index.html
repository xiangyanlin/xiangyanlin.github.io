windows下升级node.js, 在cmd 运行n stable报错bash不是内部或外部命令，也不是可运行的程序 或批处理文件。


在GitHub官网，搜索gnvm，下载


放在node.js的安装目录下


然后打开cmd命令行窗口，输入：gnvm update latest，等待更新。
当然也可以直接覆盖.

https://jingyan.baidu.com/album/9158e000fc556ea2541228e2.html?picindex=1.
